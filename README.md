# Image_Processing_Project

# Interactive-Games-with-Gesture-Detection
Hand Gesture Games using MediaPipe and PyGame
- Drawing Canvas
- Rock, Paper, Scissors

# Install and Run

```
$ pip3 install mediapipe
$ pip3 install pygame
$ python3 main.py
```
